A REST API built with Node.js, Express, and MongoDB that allows users to find coders based on their geographic location.

#Features
Full CRUD: Create, Read, Update, and Delete coder profiles.

GeoJSON Integration: Store and query coders using real-world coordinates.

Error Handling: Custom middleware to handle validation and database errors.


#Requirements to use this api:

Node.js installed on your machine.

MongoDB running locally on port 27017.

Postman or Insomnia for testing routes.

#installation
Clone the repository or navigate to your project folder.
Initialize the project by running npm init -y

Install dependencies: npm install express mongoose body-parser

Run the app by nodemon index
